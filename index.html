Demo

コンソールを開いてボタンを押すとサンプルログを確認できます。




How To Use

  1. outputLog.jsの読み込み（jQueryを一番上で読み込み、そのすぐ下でoutputLog.jsを読み込んでください。）

    <script src="【jQueryのsrc】"></script><!-- jQueryを読み込み -->
<script type="text/javascript" src="./outputLog.js"></script><!-- outputLog.jsを読み込み -->

  2. outputLog.jsの仕様

    js内に配置したconsole.log()などの内容が記録され、コンソールにも現在時刻付きでログが表示される。
    画面右上の「ろぐ」ボタンを押下すると記録したログの一覧をテキストファイルでダウンロードできる。

    ★outputLog.jsの対象
    • console.log()
    • console.info()
    • console.warn()
    • console.error()

  3. outputLog.jsの操作

    ブラウザのコンソールから下記のコードを実行できる。
    // ライブラリを有効化する（outputLog.jsを読み込んだ時点ですでに有効化されている）
outputLog.enable();

// ライブラリを無効化する
outputLog.disable();

// 記録しているlogを消去する
outputLog.reset();

// 記録しているlogをダウンロードする（「ろぐ」ボタン押下と同義）
outputLog.downLoad();

  4. outputLog.jsのダウンロード
